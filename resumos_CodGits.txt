Funções Git

git clone -> copiar o repositorio no computador pessoal


git add -> incluir o arquivo modificado na lista para atualizações


git commit -m  -> salvar e comentar sobre as modificações feitas 


git pull -> verifica se a cópia local está atualizada e se o virtual foi alterado.


git push -> atualiza o repositório virtual com as modificações locais 



Funções no Windowns

"cd.." e "cd" -> comandos para voltar pastas e selecionar pastas no Windowns. 

dir -> verificar se os arquivos foram copiados para o note na pasta no local indicado

echo -> criar arquivos pelo prompt
